Godot: Web-Based Multiplayer

(Click to view my write-up on the official Godot blog.)

Godot is a free and open source game engine. I was contracted by Google to improve their multiplayer features

I added support to the engine for a web protocol called WebRTC. A simple interface for WebRTC is key for web games because it can provide the lowest latency form of communication available on the web.

These contributions also gave Godot a straightforward way to make peer-to-peer multiplayer games.
